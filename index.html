Description

On the map, there are different types of tiles that you need to use wisely to design the railway network. Each tile has its own rules that influence where and how you can build the railway. There are four possible tile types:

Empty tile

Bridge tile

Mountain tile

Oasis tile

Empty Cells: You can place any type of track. Bridges: Only straight tracks can be placed. Mountains: Only curved tracks can be placed. Oasis: No tracks can be placed.

The goal is to build a continuous circular railway that visits every accessible tile without crossing itself, branching out, or passing through any tile more than once.

Select a tile you want to place the track on. Use the up and down arrows to change track types. Use the left and right arrows to rotate the tracks. Alternatively, you can use the icons beside the track for these actions. Place a track by pressing Enter or by clicking on the image.

Try it for yourself!

The game continues until you create a continuous path. If you place all tracks without forming a connected path, a 1-minute penalty will be added to your time.
